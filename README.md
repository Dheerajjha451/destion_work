# Destion Innovations

### Setup
1. Clone the repository:

   ```bash
   git clone https://github.com/Dheerajjha451/destion_work

2. Navigate into the project directory:
    ```bash
    cd destion_work
    
3. Install the required dependencies:

```
npm install
```
### Create .env file

```
AUTH_SECRET=""
RESEND_API_KEY=
NEXT_PUBLIC_APP_URL=
DATABASE_URL=""

```
### Start the project

```
npm run dev
```
